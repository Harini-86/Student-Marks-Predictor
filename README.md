# 🎓 Student Marks Predictor

## 📌 Project Overview

This is a Machine Learning project that predicts a student's marks based on the number of hours they study using the Linear Regression algorithm.

---

## 📊 Dataset

The dataset contains two columns:

- Hours
- Marks

Example:

| Hours | Marks |
|-------|-------|
| 1 | 20 |
| 2 | 30 |
| 3 | 40 |

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 🤖 Machine Learning Algorithm

- Linear Regression

---

## 📈 Model Performance

R² Score: **0.9644**

---

## 🚀 Features

- Load dataset
- Visualize data
- Train Linear Regression model
- Predict student marks
- Save trained model using Pickle
